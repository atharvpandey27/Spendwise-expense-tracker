# Spendwise – Expense Tracker

A personal finance tracker web app to manage income and expenses with visual charts and category breakdowns.

## Features
- Add income and expense transactions
- Categorize transactions (Food, Transport, Education, Shopping, Health, etc.)
- Real-time balance, income, and expense summary
- 7-day bar chart visualization
- Category-based color coding
- Persistent data via localStorage

## Tech Stack
`HTML5` `CSS3` `JavaScript (ES6+)` `localStorage API` `CSS Grid` `CSS Animations`

## How to Run
1. Clone the repository
2. Open `index.html` in a browser — no server or dependencies needed

## Key Concepts Demonstrated
- **Data Persistence**: All transactions saved via `localStorage`
- **Dynamic DOM Manipulation**: Real-time UI updates on every action
- **Data Aggregation**: Income/expense totals computed from transaction array
- **Data Visualization**: Bar chart built purely with CSS height calculations
- **Responsive Layout**: CSS Grid with mobile breakpoints
